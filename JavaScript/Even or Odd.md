**Problem:**

Create a function that takes an integer as an argument and returns `"Even"` for even numbers or `"Odd"` for odd numbers.

**Solution:**

```javascript

function even_or_odd(number) {
  return !(number%2) ? "Even" : "Odd";
}

```
